# HR - Documents

HR users focus on strategic project visibility, deadlines, client/project records, reports, documents, and people/workload insight.

![HR Documents](/docs/screenshots/hr/documents.png)

## When To Use This Page

Use this page when you need to complete documents work as the HR role. The screenshot above was captured while signed in as this role, so the visible navigation and actions reflect the permissions available to that panel.

## Step By Step

1. Open Documents from the sidebar.
2. Use search/filter controls to locate the file before uploading a duplicate.
3. Select Upload, choose the file, and enter a clear document name/category.
4. After upload, verify the document card/list item shows the expected status.
5. Approvers should view the document before approving or rejecting it.
6. Use download/view/delete only when the current document and permission level are correct.

## Checks Before You Leave

1. Confirm the saved state appears on screen.
2. Refresh or revisit the page if the action affects counts, dashboards, or project boards.
3. Notify the next responsible person if the workflow now depends on them.

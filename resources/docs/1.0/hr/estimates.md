# HR - Estimates

HR users focus on strategic project visibility, deadlines, client/project records, reports, documents, and people/workload insight.

![HR Estimates](/docs/screenshots/hr/estimates.png)

## When To Use This Page

Use this page when you need to complete estimates work as the HR role. The screenshot above was captured while signed in as this role, so the visible navigation and actions reflect the permissions available to that panel.

## Step By Step

1. Open Estimates from the sidebar.
2. Search or filter to locate an existing estimate before creating a duplicate.
3. Create or open an estimate and confirm client, currency, items, tax/discount, and total.
4. Use linked project or conversion actions only after the estimate details are approved.
5. If Xero integration is visible, confirm connection status before syncing or exporting.
6. Save and verify the estimate status and totals in the list.

## Checks Before You Leave

1. Confirm the saved state appears on screen.
2. Refresh or revisit the page if the action affects counts, dashboards, or project boards.
3. Notify the next responsible person if the workflow now depends on them.

# HR - Emails

HR users focus on strategic project visibility, deadlines, client/project records, reports, documents, and people/workload insight.

![HR Emails](/docs/screenshots/hr/emails.png)

## When To Use This Page

Use this page when you need to complete emails work as the HR role. The screenshot above was captured while signed in as this role, so the visible navigation and actions reflect the permissions available to that panel.

## Step By Step

1. Open Emails from the sidebar.
2. Select the mailbox/folder if multiple folders are available.
3. Use search and message preview to find the relevant client or project conversation.
4. Open a message to inspect sender, recipients, body, and attachments.
5. Use compose/reply only after selecting the correct account and recipients.
6. Attach files carefully and verify the sent message appears in the correct thread or sent folder.

## Checks Before You Leave

1. Confirm the saved state appears on screen.
2. Refresh or revisit the page if the action affects counts, dashboards, or project boards.
3. Notify the next responsible person if the workflow now depends on them.

# Login

Use this entry before any role-specific manual.

![Aura login](/docs/screenshots/login.png)

## Step By Step

1. Open the Aura application URL in a browser.
2. Enter your company email address in the Email field.
3. Enter your password in the Password field.
4. Select Sign In and wait until the dashboard/sidebar appears.
5. If two-factor authentication is enabled, complete the verification challenge before continuing.
6. If the login fails, check that the account is active and contact an administrator before retrying.

## Troubleshooting

1. If credentials fail, verify the account is active in Users.
2. If the page loops back to login, clear browser storage and sign in again.
3. If two-factor authentication is enabled, use the current authenticator code or recovery workflow.
4. If a user sees the wrong panel, verify their database role and department assignment.

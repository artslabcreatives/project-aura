# Account Manager - Project Board

Account managers focus on assigned client-facing work, review queues, estimates, reports, and cross-functional follow-up.

![Account Manager Project Board](/docs/screenshots/account-manager/project-board.png)

## When To Use This Page

Use this page when you need to complete project board work as the Account Manager role. The screenshot above was captured while signed in as this role, so the visible navigation and actions reflect the permissions available to that panel.

## Step By Step

1. Open a project from the sidebar Projects area.
2. Review the board columns/stages from left to right before changing tasks.
3. Open a task card to inspect details, assignees, attachments, comments, history, and time logs.
4. Move or update tasks according to the workflow rules for your role.
5. Add tasks, stages, attachments, reports, PO details, or campaign files only when the controls are visible and the project is active.
6. After saving, confirm the task/project appears in the correct stage and that related counts updated.

## Checks Before You Leave

1. Confirm the saved state appears on screen.
2. Refresh or revisit the page if the action affects counts, dashboards, or project boards.
3. Notify the next responsible person if the workflow now depends on them.

# Account Manager - Review Needed

Account managers focus on assigned client-facing work, review queues, estimates, reports, and cross-functional follow-up.

![Account Manager Review Needed](/docs/screenshots/account-manager/review-needed.png)

## When To Use This Page

Use this page when you need to complete review needed work as the Account Manager role. The screenshot above was captured while signed in as this role, so the visible navigation and actions reflect the permissions available to that panel.

## Step By Step

1. Open Review Needed from the sidebar.
2. Scan the list for the most urgent task or project awaiting review.
3. Open the item and inspect description, attachments, comments, due date, and stage context.
4. Approve only when the work meets the required standard and target stage is correct.
5. Request revision with a clear comment when the task needs rework.
6. Verify the item leaves the review list or changes status after your action.

## Checks Before You Leave

1. Confirm the saved state appears on screen.
2. Refresh or revisit the page if the action affects counts, dashboards, or project boards.
3. Notify the next responsible person if the workflow now depends on them.

# Admin - Clients

Admins operate across the full Aura workspace: users, projects, tasks, clients, estimates, reports, documents, SSO applications, AI scenarios, and cross-department analytics.

![Admin Clients](/docs/screenshots/admin/clients.png)

## When To Use This Page

Use this page when you need to complete clients work as the Admin role. The screenshot above was captured while signed in as this role, so the visible navigation and actions reflect the permissions available to that panel.

## Step By Step

1. Open Clients from the sidebar.
2. Search for the company before creating a new client record.
3. Open a client profile to review contacts, estimates, invoices, financial summary, and history.
4. Add or update contacts with accurate names, titles, emails, and phone numbers.
5. Use merge/delete carefully; confirm the correct company record is selected.
6. Return to the profile after saving to verify the visible client details are correct.

## Checks Before You Leave

1. Confirm the saved state appears on screen.
2. Refresh or revisit the page if the action affects counts, dashboards, or project boards.
3. Notify the next responsible person if the workflow now depends on them.

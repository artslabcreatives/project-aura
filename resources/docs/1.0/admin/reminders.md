# Admin - Reminders

Admins operate across the full Aura workspace: users, projects, tasks, clients, estimates, reports, documents, SSO applications, AI scenarios, and cross-department analytics.

![Admin Reminders](/docs/screenshots/admin/reminders.png)

## When To Use This Page

Use this page when you need to complete reminders work as the Admin role. The screenshot above was captured while signed in as this role, so the visible navigation and actions reflect the permissions available to that panel.

## Step By Step

1. Open Reminders from the sidebar.
2. Review active reminders before creating a new one to avoid duplicates.
3. Enter a clear title, optional description, and the exact reminder date/time.
4. Save the reminder and confirm it appears in the active list.
5. Use edit/delete actions to correct or remove reminders that are no longer needed.
6. For automated reminder controls, verify recipient and schedule settings before saving.

## Checks Before You Leave

1. Confirm the saved state appears on screen.
2. Refresh or revisit the page if the action affects counts, dashboards, or project boards.
3. Notify the next responsible person if the workflow now depends on them.

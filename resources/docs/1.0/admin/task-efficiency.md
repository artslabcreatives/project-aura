# Admin - Task Efficiency

Admins operate across the full Aura workspace: users, projects, tasks, clients, estimates, reports, documents, SSO applications, AI scenarios, and cross-department analytics.

![Admin Task Efficiency](/docs/screenshots/admin/task-efficiency.png)

## When To Use This Page

Use this page when you need to complete task efficiency work as the Admin role. The screenshot above was captured while signed in as this role, so the visible navigation and actions reflect the permissions available to that panel.

## Step By Step

1. Open the efficiency page from the sidebar.
2. Select the person, department, or date range if controls are visible.
3. Review completion rate, average completion time, overdue counts, and trend indicators.
4. Compare the metric with the underlying task/project list before making a staffing decision.
5. Export or screenshot the result when it needs to be included in an operational report.
6. Revisit the page after task status changes to confirm the metric updated.

## Checks Before You Leave

1. Confirm the saved state appears on screen.
2. Refresh or revisit the page if the action affects counts, dashboards, or project boards.
3. Notify the next responsible person if the workflow now depends on them.

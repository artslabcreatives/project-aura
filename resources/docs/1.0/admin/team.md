# Admin - Team

Admins operate across the full Aura workspace: users, projects, tasks, clients, estimates, reports, documents, SSO applications, AI scenarios, and cross-department analytics.

![Admin Team](/docs/screenshots/admin/team.png)

## When To Use This Page

Use this page when you need to complete team work as the Admin role. The screenshot above was captured while signed in as this role, so the visible navigation and actions reflect the permissions available to that panel.

## Step By Step

1. Open Team from the sidebar.
2. Search or scan by department to find the person you need.
3. Open a member card to review role, department, status, and workload details.
4. For roles with permission, use Add/Edit actions to maintain member details.
5. When changing a role or department, verify downstream visibility and permissions before saving.
6. Use the page as the source of truth before assigning work to a person.

## Checks Before You Leave

1. Confirm the saved state appears on screen.
2. Refresh or revisit the page if the action affects counts, dashboards, or project boards.
3. Notify the next responsible person if the workflow now depends on them.

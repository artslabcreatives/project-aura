# Aura Manuals Overview

These manuals document Aura from each working panel separately. Every frontend role has its own section, every captured page has its own entry, and backend/developer material is separated from employee-facing instructions.

![Aura login](/docs/screenshots/login.png)

## How To Use These Docs

1. Start with the role that matches the user you are training.
2. Open the role overview, then follow each page entry in order.
3. Use the screenshot at the top of each page to confirm the user is in the correct screen.
4. Follow the step-by-step workflow exactly for first-time training.
5. Use the Backend Admin section only for Filament administration.
6. Use the Developer section only for implementation, API, deployment, and troubleshooting work.

# Employee - Profile

Employees focus on their own assignments, collaboration, reminders, reports, documents, and personal configuration.

![Employee Profile](/docs/screenshots/employee/profile.png)

## When To Use This Page

Use this page when you need to complete profile work as the Employee role. The screenshot above was captured while signed in as this role, so the visible navigation and actions reflect the permissions available to that panel.

## Step By Step

1. Open Profile from the avatar/user menu.
2. Review your name, email, role, department, and account status.
3. Update available profile details or avatar if the controls are visible.
4. Use password or two-factor controls from the security card where available.
5. Confirm profile changes by refreshing or reopening the user menu.
6. Sign out from the account menu when work is complete on a shared machine.

## Checks Before You Leave

1. Confirm the saved state appears on screen.
2. Refresh or revisit the page if the action affects counts, dashboards, or project boards.
3. Notify the next responsible person if the workflow now depends on them.

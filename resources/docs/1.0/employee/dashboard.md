# Employee - Dashboard

Employees focus on their own assignments, collaboration, reminders, reports, documents, and personal configuration.

![Employee Dashboard](/docs/screenshots/employee/dashboard.png)

## When To Use This Page

Use this page when you need to complete dashboard work as the Employee role. The screenshot above was captured while signed in as this role, so the visible navigation and actions reflect the permissions available to that panel.

## Step By Step

1. Open Dashboard from the sidebar or visit the home page.
2. Read the summary cards first; they show workload, active work, overdue work, and completion signals for the role.
3. Review charts, calendars, or lists below the cards to identify the next action.
4. Select any clickable card, project, task, or calendar item to drill into details.
5. Use the global search if you need a specific record instead of browsing lists.
6. Refresh the page if real-time updates appear stale after changing roles or opening the page for a long time.

## Checks Before You Leave

1. Confirm the saved state appears on screen.
2. Refresh or revisit the page if the action affects counts, dashboards, or project boards.
3. Notify the next responsible person if the workflow now depends on them.

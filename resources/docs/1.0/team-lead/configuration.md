# Team Lead - Configuration

Team leads focus on department delivery, task review, workload visibility, and project execution for their team.

![Team Lead Configuration](/docs/screenshots/team-lead/configuration.png)

## When To Use This Page

Use this page when you need to complete configuration work as the Team Lead role. The screenshot above was captured while signed in as this role, so the visible navigation and actions reflect the permissions available to that panel.

## Step By Step

1. Open Configuration from the profile menu or direct route.
2. Review Appearance, Notifications, Security, and Integrations sections.
3. Change one setting at a time so the saved result is easy to confirm.
4. For notification settings, allow browser permission when prompted if push notifications are required.
5. For security changes, verify the current password and store recovery details securely.
6. Return to the relevant page and confirm the changed preference is reflected.

## Checks Before You Leave

1. Confirm the saved state appears on screen.
2. Refresh or revisit the page if the action affects counts, dashboards, or project boards.
3. Notify the next responsible person if the workflow now depends on them.

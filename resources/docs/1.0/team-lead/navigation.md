# Team Lead - After Login Navigation

Team leads focus on department delivery, task review, workload visibility, and project execution for their team.

![Team Lead After Login Navigation](/docs/screenshots/team-lead/after-login.png)

## When To Use This Page

Use this page when you need to complete after login navigation work as the Team Lead role. The screenshot above was captured while signed in as this role, so the visible navigation and actions reflect the permissions available to that panel.

## Step By Step

1. Start from the left sidebar after login.
2. Use the top search box to find projects, tasks, clients, comments, and other indexed records.
3. Use the notification bell to review unread updates.
4. Use the profile avatar for account settings, configuration, and sign out.
5. Open a sidebar page once, wait for the table/cards to finish loading, and then begin work.
6. For project work, expand the Projects area and select the project or stage you need.

## Checks Before You Leave

1. Confirm the saved state appears on screen.
2. Refresh or revisit the page if the action affects counts, dashboards, or project boards.
3. Notify the next responsible person if the workflow now depends on them.
